# Smart City Analytics Dashboard

## 📌 Overview
This project focuses on using data analytics to improve urban services such as transportation, waste management, and energy efficiency.

## 📊 Tools Used
- Power BI
- Excel
- Python (for data cleaning)
- Power Query Editor
- DAX

## 📁 Dataset
The dataset includes simulated data for:
- Transportation (traffic volume, speed)
- Waste Management (waste collected, bin levels)
- Energy (consumption, peak load)

## ⚙️ Data Processing
- Removed duplicates
- Handled missing values using averages
- Standardized inconsistent data

## 📈 Dashboard Features
- KPI Cards:
  - Average Traffic Speed
  - Total Waste Collected
  - Total Energy Consumption
- Visualizations:
  - Traffic Trends (Line Chart)
  - Waste by Area (Bar Chart)
  - Energy Distribution (Pie Chart)

## 🔍 Key Insights
- Traffic peaks in April and is lowest in February
- Kothrud has highest waste and energy consumption
- High overall energy and waste indicates need for optimization

## 🎯 Outcome
This project demonstrates how data-driven insights can improve urban planning, resource allocation, and operational efficiency in smart cities.
